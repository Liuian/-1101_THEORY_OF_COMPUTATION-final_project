# line bot-穿搭顏色心理學(計算理論final project)
## 前言
此linebot旨在幫助人們根據穿衣言色心裡學在適當的場合穿著更適合的衣服顏色（ex : 談判時做好穿著暖色系的衣服，因為這會給人較有重量級安定的形象，能幫助談判的過程更順利！），同時也可以幫助人們根據對方的穿著顏色分析對方的心理狀態。
